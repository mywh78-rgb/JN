# 跨境电商竞品与市场调研 SOP

本文档用于规范跨境电商项目在选品、供应链核价、商品上架、主图/A+详情图、短视频脚本、达人 BD 和运营复盘之前的市场与竞品调研流程。

## 一、文档用途

本 SOP 用于帮助运营人员在做商品决策前，系统收集竞品、平台、内容、价格、评论、视觉和合规信息，避免只凭感觉选品、定价、做图、写脚本或联系达人。

使用目标：

- 明确商品所属类目、结构、使用场景和目标人群。
- 判断市场需求、内容传播潜力和平台适配度。
- 拆解竞品标题、关键词、图片、视频、价格和评论。
- 提炼用户痛点、核心卖点和差异化表达。
- 输出可直接交给选品、上架、图片、视频、达人和供应链模块使用的资料。
- 提前识别侵权、禁词、虚假宣传、资质和平台审核风险。

## 二、适用场景

本 SOP 适用于以下场景：

- 新品调研
- 参考图/产品图识别
- TikTok Shop 测品前调研
- Amazon Listing 上架前调研
- Temu / Walmart / Etsy / Shopify 平台竞品调研
- 商品主图和 A+详情图制作前调研
- TikTok 短视频脚本和 AI 视频提示词制作前调研
- 达人 BD 和样品寄送前调研
- 供应链核价和利润测算前调研
- 商品测试复盘和店铺周报前的数据补充

## 三、参考图/产品图识别流程

收到参考图、产品图或供应商图片后，应先完成基础识别，再进入平台调研。

| 识别项目 | 检查内容 | 记录 |
| --- | --- | --- |
| 产品类目 | 商品属于什么品类，平台可能放在哪个类目 | 待填写 |
| 产品结构 | 主体、配件、包装、尺寸、功能结构 | 待填写 |
| 颜色/材质 | 颜色、面料、材质、纹理、透明度、质感 | 待填写 |
| 款式/规格 | 尺码、容量、套装、数量、变体 | 待填写 |
| 使用方式 | 谁使用、怎么使用、在哪些场景使用 | 待填写 |
| 目标人群 | 年龄、性别、兴趣、消费场景 | 待填写 |
| 确定信息 | 图片中能够明确确认的信息 | 待填写 |
| 待补充信息 | 图片中无法确认、需要供应商补充的信息 | 待填写 |
| 潜在风险 | 侵权、敏感类目、夸大功效、资质要求 | 待填写 |

注意事项：

- 不得凭空补充产品不存在的功能。
- 不得改变产品结构、颜色、印花、配件和包装。
- 如果图片信息不足，应标记为“待确认”，不要直接下结论。
- 对宠物、泳装、美妆、母婴、医疗、安全、防护等类目应额外检查合规风险。

## 四、TikTok Shop 竞品调研

TikTok Shop 调研重点是内容传播、自然流潜力、达人适配和转化路径。

| 调研项目 | 重点观察 | 记录 |
| --- | --- | --- |
| 同类商品 | 是否已有同类商品在销售 | 待填写 |
| 价格带 | 主流售价、折扣价、券后价 | 待填写 |
| 内容形式 | 短视频、直播、达人内容、UGC | 待填写 |
| 3 秒 Hook | 开头是否有痛点、反差、动作或视觉冲击 | 待填写 |
| 评论反馈 | 用户问什么、抱怨什么、夸什么 | 待填写 |
| 挂车方式 | 视频挂车、达人橱窗、直播间承接 | 待填写 |
| 达人适配 | 哪类达人更适合推广 | 待填写 |
| 风险点 | 禁词、夸大功效、敏感画面、平台审核 | 待填写 |

输出重点：

- 商品是否适合 TikTok 自然流。
- 是否有清晰的一句话卖点。
- 是否容易拍出前 3 秒 Hook。
- 是否适合达人讲解和真实使用场景。
- 是否需要调整价格、佣金、视频结构或页面素材。

## 五、Amazon 竞品调研

Amazon 调研重点是搜索需求、标题关键词、图片结构、评论痛点、价格带和长期款潜力。

| 调研项目 | 重点观察 | 记录 |
| --- | --- | --- |
| 搜索关键词 | 核心词、长尾词、场景词、属性词 | 待填写 |
| 标题结构 | 品类词、卖点、规格、适用对象 | 待填写 |
| 图片结构 | 主图、卖点图、尺寸图、场景图、对比图 | 待填写 |
| A+详情页 | 材质、功能、场景、品牌故事、对比模块 | 待填写 |
| 价格带 | 低价、中价、高价竞品区间 | 待填写 |
| 评论痛点 | 差评集中问题、退货原因、用户期待 | 待填写 |
| 评分与评价数 | 竞争强度和进入难度 | 待填写 |
| 变体策略 | 颜色、尺码、套装、容量、数量组合 | 待填写 |
| 合规风险 | 认证、授权、侵权、功效表达 | 待填写 |

输出重点：

- 该商品是否适合长期沉淀搜索流量。
- 标题和关键词应该如何组织。
- 主图和 A+详情图应该优先表达哪些信息。
- 评论痛点是否能转化为差异化卖点。
- 是否存在过强竞争或低利润风险。

## 六、Temu / Walmart / Etsy / Shopify 竞品调研

不同平台的调研重点不同，不能用同一套标准判断所有商品。

| 平台 | 调研重点 | 输出判断 |
| --- | --- | --- |
| Temu | 低价竞争力、供货价、物流体积、同质化程度 | 是否适合低价走量，是否还能保留利润 |
| Walmart | 标准化商品、价格稳定性、履约能力、评价质量 | 是否适合稳定上架和长期履约 |
| Etsy | 手工感、设计感、个性化、礼品属性 | 是否适合差异化表达和高溢价 |
| Shopify | 品牌故事、广告承接、内容素材、复购空间 | 是否适合独立站转化和品牌化经营 |

调研时应记录：

- 竞品链接和平台。
- 价格带和促销方式。
- 主图/详情图风格。
- 标题关键词和卖点结构。
- 用户评论和痛点。
- 履约、退货和售后风险。
- 是否适合当前团队资源执行。

## 七、Google Trends / Reddit / Pinterest / YouTube Shorts 趋势观察

趋势观察用于补充平台内竞品调研，但不能代替实际商品数据判断。

| 渠道 | 观察重点 | 注意事项 |
| --- | --- | --- |
| Google Trends | 搜索趋势、季节性、地区热度 | 不等于销量，只做需求方向参考 |
| Reddit | 用户真实讨论、抱怨、使用场景、替代方案 | 不直接复制用户内容，不编造热度 |
| Pinterest | 视觉风格、场景趋势、礼品/家居/时尚灵感 | 适合图片和场景方向参考 |
| YouTube Shorts | 短视频结构、Hook、使用演示、评论反馈 | 适合补充 TikTok 内容思路 |

记录建议：

- 关键词或话题。
- 观察到的用户需求。
- 高频痛点或问题。
- 视觉风格趋势。
- 内容表达方式。
- 对产品卖点、图片、视频或达人 brief 的启发。

## 八、竞品标题与关键词拆解

标题和关键词拆解用于商品上架、图片文案、短视频脚本和达人话术。

| 拆解项目 | 说明 | 记录 |
| --- | --- | --- |
| 核心品类词 | 商品是什么 | 待填写 |
| 使用场景词 | 用在什么场景 | 待填写 |
| 适用对象词 | 谁会购买或使用 | 待填写 |
| 核心卖点词 | 解决什么问题或突出什么优势 | 待填写 |
| 规格属性词 | 尺寸、容量、数量、材质、颜色 | 待填写 |
| 情绪/礼品词 | cute、cozy、gift、travel 等 | 待填写 |
| 不建议使用词 | 夸大、绝对化、医疗化或侵权表达 | 待填写 |

输出应包含：

- 可用于商品标题的关键词。
- 可用于短视频字幕和口播的口语化表达。
- 可用于达人 brief 的卖点话术。
- 需要避免的禁词和高风险表达。

## 九、主图/A+详情图视觉拆解

视觉拆解用于指导 `product-image-brief.md` 的图片规划。

| 图片类型 | 观察重点 | 输出方向 |
| --- | --- | --- |
| 主图 | 主体大小、背景、角度、颜色、第一眼点击理由 | 主图应该突出什么 |
| 细节图 | 材质、结构、工艺、配件、包装 | 哪些细节必须拍清楚 |
| 场景图 | 使用环境、人物/宠物互动、情绪氛围 | 哪些场景最能带来购买理由 |
| 尺寸图 | 尺寸、比例、容量、穿着/摆放效果 | 哪些规格信息需要视觉化 |
| 对比图 | 新旧对比、竞品对比、使用前后 | 是否适合做对比表达 |
| A+详情图 | 卖点顺序、功能解释、品牌感、模块结构 | A+图如何组织信息 |

注意事项：

- 不要照抄竞品图片和排版。
- 不要使用未授权品牌、人物、素材或图案。
- 不要做九宫格、拼图、合集图或长图合集。
- 商品结构、颜色、印花、肩带、配件、包装等必须准确还原。

## 十、短视频内容结构拆解

短视频拆解用于指导 `tiktok-video-script-sop.md` 和达人 brief。

| 拆解项目 | 观察重点 | 记录 |
| --- | --- | --- |
| 前 3 秒 Hook | 痛点、反差、动作、画面冲击 | 待填写 |
| 产品出现时机 | 开头出现还是中段出现 | 待填写 |
| 使用场景 | 家庭、户外、办公室、旅行、宠物互动等 | 待填写 |
| 卖点表达 | 功能、材质、尺寸、便利性、情绪价值 | 待填写 |
| 字幕/口播 | 是否短句化、英文口语化、易读 | 待填写 |
| 镜头节奏 | 每 2-3 秒是否有变化 | 待填写 |
| CTA | 是否自然引导点击、收藏、购买 | 待填写 |
| 评论反馈 | 用户是否询问价格、链接、尺码、效果 | 待填写 |

输出应包含：

- 适合 15 秒、20 秒、30 秒视频的脚本方向。
- 可复用的 Hook 类型。
- 适合 Seedance / Dreamina 的提示词素材。
- 需要避免的夸大、医疗、安全或绝对化表达。

## 十一、用户评论痛点提炼

用户评论是卖点、图片、视频和页面优化的重要来源。

| 评论类型 | 需要提炼的内容 | 记录 |
| --- | --- | --- |
| 好评 | 用户为什么满意，最喜欢什么 | 待填写 |
| 差评 | 质量、尺寸、物流、颜色、预期不符等问题 | 待填写 |
| 问答 | 用户购买前最关心的问题 | 待填写 |
| 退货原因 | 尺码、破损、误解、材质、功能不符 | 待填写 |
| 使用场景 | 用户实际怎么用，在哪用，给谁用 | 待填写 |
| 未被满足需求 | 竞品没有解决的问题 | 待填写 |

提炼方式：

- 将评论转化为痛点，而不是只复制原句。
- 区分真实高频问题和少量个案。
- 避免编造销量、GMV、热搜词、平台数据和评论结论。
- 将痛点映射到标题、图片、视频、详情页和客服说明。

## 十二、核心卖点提炼

卖点应从用户痛点、平台场景和商品真实能力中提炼。

| 卖点类型 | 示例方向 | 记录 |
| --- | --- | --- |
| 功能卖点 | 解决具体问题，提升便利性 | 待填写 |
| 材质卖点 | 面料、触感、耐用性、轻便性 | 待填写 |
| 场景卖点 | 旅行、家庭、宠物、运动、礼品 | 待填写 |
| 情绪卖点 | cute、cozy、confidence、giftable 等 | 待填写 |
| 差异化卖点 | 竞品没有讲清楚或没有做到的点 | 待填写 |
| 内容卖点 | 适合短视频前 3 秒展示的点 | 待填写 |

判断标准：

- 是否真实存在。
- 是否能被图片或视频展示。
- 是否能被美国消费者快速理解。
- 是否有合规风险。
- 是否能支持价格和利润空间。

## 十三、价格带与利润空间判断

价格调研应和供应链核价、平台费用、物流成本、达人佣金和售后风险一起判断。

| 项目 | 记录 |
| --- | --- |
| 竞品最低价 | 待填写 |
| 竞品主流价 | 待填写 |
| 竞品高价区间 | 待填写 |
| 目标售价 | 待填写 |
| 预估供货价 | 待填写 |
| 预估物流成本 | 待填写 |
| 平台费用/佣金 | 待填写 |
| 达人佣金或样品成本 | 待填写 |
| 预估毛利额 | 待填写 |
| 预估毛利率 | 待填写 |
| 价格风险 | 待填写 |

输出判断：

- 是否适合低价走量。
- 是否适合中价差异化。
- 是否适合高价品牌化。
- 是否有足够毛利支持达人、折扣、广告或退货。
- 是否需要重新压价、换供应商或暂缓上架。

## 十四、合规风险与禁词风险检查

调研阶段必须提前识别合规风险，避免后续上架、图片、视频和达人内容返工。

| 风险类型 | 检查内容 | 结果 |
| --- | --- | --- |
| 侵权风险 | 品牌、Logo、角色、图案、专利、外观设计 | 待填写 |
| 禁售/限售 | 平台是否允许销售，是否需要资质 | 待填写 |
| 资质认证 | FDA、CPC、FCC、CE、UL 或其他要求 | 待填写 |
| 医疗功效 | 治疗、治愈、消炎、止痛、抗菌等表达 | 待填写 |
| 安全承诺 | 100% safe、guaranteed、risk-free 等绝对表达 | 待填写 |
| 虚假宣传 | 夸大效果、无法验证的数据或结果 | 待填写 |
| 未授权素材 | 竞品图、达人图、平台图、明星图 | 待填写 |
| 敏感画面 | 裸露、低俗、危险动作、儿童/宠物不当场景 | 待填写 |

处理建议：

- 高风险商品进入暂缓或进一步资质确认。
- 高风险词汇进入禁用表达清单。
- 视觉和视频 brief 中明确禁止风险画面和素材。
- 达人 brief 中提前写清楚不能说、不能拍、不能承诺的内容。
