# TikTok Shop 日常运营 SOP

本文档用于记录 TikTok Shop 店铺的日常运营检查流程。当前版本为基础版，适合后续根据真实业务继续补充细节。

## 1. 店铺日常检查

每日开始运营前，先确认店铺整体状态是否正常。

- 检查店铺后台是否可以正常登录。
- 查看店铺是否存在平台通知、违规提醒或待处理事项。
- 检查店铺评分、客服表现、履约表现等核心指标是否异常。
- 查看是否有活动报名、平台任务或限时要求需要处理。
- 记录当天需要优先跟进的问题。

## 2. 商品状态检查

确认商品是否处于可售、合规、信息完整的状态。

- 检查重点商品是否正常上架。
- 查看是否有商品被下架、审核中、库存不足或价格异常。
- 检查商品标题、主图、详情页、规格和价格是否准确。
- 确认热销商品库存是否充足。
- 对异常商品记录原因，并安排修正。

## 3. 订单与履约检查

确保订单可以按时处理，降低超时和差评风险。

- 查看新增订单、待发货订单和异常订单。
- 检查是否有即将超时的订单。
- 确认物流单号、发货状态和履约时效。
- 跟进取消、退款、退货或买家投诉。
- 对异常订单记录处理进度和责任人。

## 4. 内容与短视频检查

维护店铺内容节奏，持续优化商品曝光。

- 检查当天计划发布的视频或内容是否准备完成。
- 查看已发布短视频的数据表现，例如播放、点击、转化和互动。
- 记录表现较好的内容选题和商品组合。
- 检查视频素材是否符合平台规则。
- 根据数据表现调整后续内容计划。

## 5. 达人/联盟跟进

保持达人合作和联盟推广的持续推进。

- 查看是否有达人申请、合作邀约或待回复消息。
- 跟进样品寄送、内容发布时间和视频表现。
- 记录优质达人、潜力达人和需要停止合作的达人。
- 检查联盟商品佣金、推广状态和合作效果。
- 汇总达人合作问题，方便后续复盘。

## 6. 风险与合规提醒

运营过程中优先避免账号、商品和内容风险。

- 不上传侵权、虚假宣传或平台禁售商品。
- 不使用未经授权的品牌、图片、音乐或素材。
- 不夸大产品功效，不承诺无法保证的结果。
- 定期查看平台规则更新和政策通知。
- 对违规提醒、警告和处罚及时记录并处理。

## 每日记录建议

每日运营结束后，建议记录以下内容：

- 今日完成的重点事项
- 发现的问题和处理进度
- 需要明天继续跟进的事项
- 商品、订单、内容或达人方面的异常情况
- 可沉淀为 SOP 的新经验
