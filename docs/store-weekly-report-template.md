# 店铺周报模板

本文档用于每周复盘 TikTok Shop、Amazon、Shopify、Temu 等平台的店铺运营表现，帮助团队把选品、供应链、上架、素材、内容、达人、订单、日常运营和商品测试复盘串联为一份周度经营报告。

## 一、文档用途

本模板用于记录店铺每周核心数据、关键问题、业务进度和下周动作，帮助判断当前运营是否健康，以及哪些商品、内容、达人、供应链或履约环节需要优先处理。

适用目标：

- 汇总本周店铺经营表现
- 复盘商品、内容、达人、订单、利润和风险
- 跟踪选品、测品、上架和供应链进度
- 沉淀每周问题清单和下周重点动作
- 为月报、商品复盘和团队协作提供依据

## 二、适用场景

本模板适用于以下场景：

- TikTok Shop 店铺周度运营复盘
- Amazon 店铺 Listing、订单和利润复盘
- Shopify 独立站商品、流量和转化复盘
- Temu 商品价格、履约和售后复盘
- 跨平台商品测试和运营周会
- 运营、供应链、内容、达人和客服团队同步
