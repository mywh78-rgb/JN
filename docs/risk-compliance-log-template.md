# 风险合规记录模板

## 文档用途

本文档用于记录跨境电商运营中的商品、标题、图片、视频、达人内容、广告投放、平台处罚、敏感词、资质、侵权、虚假宣传和履约风险，帮助团队建立可追踪的风险处理和复盘机制。

## 适用场景

- 商品上架前合规检查
- 商品审核失败或被下架
- 平台通知违规、处罚或绩效风险
- 图片、视频、达人内容存在夸大或侵权风险
- 广告、促销或客服话术存在合规风险
- 资质、认证、品牌授权或类目限制需要确认
- 周报、月报、测品复盘需要引用风险记录

## 风险来源分类

| 来源 | 示例 |
| --- | --- |
| 商品信息 | 标题、卖点、详情页、SKU、规格、类目 |
| 图片素材 | 主图、A+图、场景图、包装图、AI 生图 |
| 视频内容 | 短视频、直播切片、英文字幕、口播、CTA |
| 达人内容 | 达人脚本、直播话术、联盟素材、样品展示 |
| 广告投放 | 广告素材、落地页、促销承诺、受众定向 |
| 平台处罚 | 下架、限流、扣分、冻结、绩效预警 |
| 供应链 | 资质、包装、标签、质检、履约能力 |
| 客服售后 | 退款承诺、评价引导、争议处理话术 |

## 风险类型

- 禁售或限售类目风险
- 品牌侵权风险
- 图片或素材版权风险
- 商标、专利、外观侵权风险
- 虚假宣传风险
- 夸大功效风险
- 医疗、安全、功效类敏感表达风险
- 资质或认证缺失风险
- 平台禁词或敏感词风险
- 履约、物流、售后绩效风险
- 达人内容失控风险
- 广告政策违规风险

## 风险等级

| 等级 | 判断标准 | 处理方式 |
| --- | --- | --- |
| 低 | 单点表达不严谨，短期影响有限 | 修改文案或素材并记录 |
| 中 | 可能影响审核、转化、评价或售后 | 暂停发布，完成复核后再上线 |
| 高 | 可能导致下架、处罚、投诉或侵权争议 | 立即暂停，升级处理，必要时停止推广 |

## 处理流程

1. 记录风险来源、页面或内容链接。
2. 截图或保存平台通知、客户反馈、达人内容或广告素材。
3. 判断风险类型和风险等级。
4. 指定处理人和截止时间。
5. 确认处理动作：修改、下架、暂停、补资质、联系达人、申诉或停止测试。
6. 记录处理结果和复盘结论。
7. 同步到周报、月报或商品测试复盘。

## 常见预防动作

- 更新标题、卖点、详情页和 FAQ。
- 替换侵权或不确定来源素材。
- 调整图片、视频、字幕和达人 brief。
- 补充资质、授权、认证或平台要求资料。
- 暂停广告、达人推广或直播活动。
- 修改客服回复模板，避免违规承诺。
- 将高风险商品降级、暂停或停止测试。
