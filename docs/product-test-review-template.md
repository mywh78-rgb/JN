# 商品测试复盘模板

本文档用于记录跨境电商商品测试后的结果复盘，帮助判断商品是否继续放量、优化后再测、转入长期运营，或暂缓/停止推进。

## 一、文档用途

本模板用于 TikTok Shop、Temu、Amazon、Shopify 等平台的商品测试复盘，统一记录商品从选品、核价、上架、图片、短视频、达人、订单、利润到合规风险的实际表现。

适用目标：

- 复盘商品测试结果
- 对比测试前判断与测试后数据
- 判断商品是否值得继续投入
- 沉淀可复用的选品、素材、达人和运营经验
- 为下一轮商品优化或放量提供依据

## 二、适用场景

本模板适用于以下场景：

- TikTok Shop 新品测品复盘
- Temu 低价商品测试复盘
- Amazon Listing 初期表现复盘
- Shopify 独立站商品测试复盘
- 达人样品合作后的商品表现复盘
- 短视频自然流内容测试复盘
- 商品上架后 7 天、14 天、30 天阶段复盘

## 十三、测试结论

根据测试结果选择一个结论：

| 结论 | 判断标准 |
| --- | --- |
| 继续放量 | 销售、利润、内容、履约和合规表现较好，可以增加投入 |
| 优化后再测 | 商品有潜力，但标题、图片、视频、价格或达人环节需要优化 |
| 转为长期款 | 商品适合长期经营，适合进入稳定上架、内容和供应链节奏 |
| 暂缓推进 | 当前利润、履约、素材或平台适配不足，需要重新评估 |
| 停止测试 | 销售差、风险高、亏损明显或供应链不可控 |
