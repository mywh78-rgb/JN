# 售后问题记录规范模板

## 文档用途

本文档用于规范跨境电商售后问题的记录、分类、跟进、复盘和预防动作沉淀，帮助团队把零散的客户反馈转化为可追踪、可复盘、可优化的运营资料。

该模板适用于 TikTok Shop、Amazon、Shopify、Temu 等平台的订单售后、退货退款、补发赔付、产品质量、页面描述、物流履约和内容误导等问题记录。

## 适用场景

- 客户发起退款、退货、取消订单或售后咨询
- 商品出现破损、少件、错发、漏发、尺寸不符或质量问题
- 商品页面、图片、短视频或达人内容引发客户误解
- 物流超时、轨迹异常、包裹丢失或履约失败
- 平台通知售后风险、绩效扣分或客户投诉
- 周报、测品复盘、供应链复盘需要引用售后问题数据

## 售后问题基础信息

每条售后问题建议至少记录以下基础信息：

- 记录日期
- 平台
- 店铺名称
- 订单编号
- 商品名称
- SKU
- 客户问题简述
- 当前处理人
- 当前状态
- 预计完成时间

## 问题来源与平台

记录售后问题的来源，有助于判断问题是否来自平台规则、客户体验、商品本身或运营内容。

常见来源包括：

- TikTok Shop 客服消息
- Amazon Buyer Message / Return Request
- Shopify 客服邮箱或在线聊天
- Temu 售后通知
- 平台绩效提醒
- 达人或联盟反馈
- 物流商反馈
- 供应商或仓库反馈
- 周报或复盘发现的问题

## 订单与商品信息

建议记录以下订单与商品字段：

- 订单编号
- 下单日期
- 发货日期
- 签收日期
- 商品链接
- SKU / 规格 / 颜色 / 尺码
- 购买数量
- 成交价格
- 运费
- 物流渠道
- 物流单号
- 是否参与活动或达人推广

## 问题类型分类

建议将售后问题归入固定分类，便于后续统计高频问题。

| 问题类型 | 说明 |
| --- | --- |
| 商品质量问题 | 破损、瑕疵、做工、材质、功能异常 |
| 商品描述不符 | 尺寸、颜色、规格、包装、配件、效果与页面不一致 |
| 物流履约问题 | 延迟、丢件、无轨迹、错派送、签收异常 |
| 发货错误 | 错发、漏发、少件、重复发货 |
| 尺码或适配问题 | 尺码偏差、不适合、使用场景不匹配 |
| 图片或视频误导 | 主图、详情图、短视频或达人内容造成误解 |
| 客户主观原因 | 不喜欢、不需要、买错、预期不一致 |
| 平台规则问题 | 退货政策、审核、绩效、争议处理 |
| 供应链问题 | 质检、包装、库存、批次差异 |
| 合规风险问题 | 侵权、禁词、夸大宣传、资质问题 |

## 客户反馈原文记录

记录客户原始反馈时，尽量保留原文，不要只写主观总结。

建议记录：

- 客户原文
- 客户提供的图片或视频证据
- 客户诉求
- 客户情绪等级
- 是否存在平台投诉风险
- 是否需要客服升级处理

## 责任归因

售后问题需要判断初步责任归因，便于后续改进。

常见归因包括：

- 商品质量问题
- 商品页面描述问题
- 图片或视频表达问题
- 尺码表或规格信息问题
- 供应商质检问题
- 仓库打包问题
- 物流渠道问题
- 客服响应问题
- 达人或内容表达问题
- 客户主观原因
- 平台规则或系统问题

## 处理流程与当前状态

建议使用固定状态追踪处理进度：

| 状态 | 说明 |
| --- | --- |
| 待确认 | 已收到问题，尚未核实订单和证据 |
| 核实中 | 正在核对订单、物流、商品和客户证据 |
| 待方案 | 问题已确认，等待确定处理方式 |
| 处理中 | 已与客户、平台、仓库或供应商推进处理 |
| 待客户确认 | 已给出方案，等待客户确认或补充资料 |
| 已解决 | 售后动作完成，客户问题关闭 |
| 需复盘 | 问题关闭但需要进入周报、测品或供应链复盘 |

## 退款、退货、补发、赔付记录

如涉及售后成本，需要记录清楚金额和原因。

建议记录：

- 是否退款
- 退款金额
- 是否退货
- 退货运费承担方
- 是否补发
- 补发 SKU 与数量
- 是否赔付
- 赔付金额或优惠券金额
- 售后总成本
- 是否影响订单利润
- 是否需要供应商承担费用

## 供应链与质检反馈

当售后问题涉及质量、包装、规格、批次或配件时，需要同步供应链侧复盘。

建议记录：

- 是否反馈供应商
- 是否涉及同批次商品
- 是否需要抽检库存
- 是否需要调整包装
- 是否需要修改质检标准
- 是否需要暂停补货
- 是否需要重新核价或压价
- 是否影响后续选品评级

## 图片、视频、页面描述问题反馈

如果售后问题来自客户预期差异，需要检查商品页面和内容素材。

建议检查：

- 标题是否准确
- 主图是否过度修饰或误导
- 详情图是否遗漏尺寸、材质、包装或使用限制
- 短视频是否夸大效果
- 英文字幕或口播是否产生误解
- 达人内容是否展示了非标配件或错误用法
- 页面是否需要补充 FAQ 或注意事项

## 达人或内容引发问题记录

当售后问题与达人、联盟、短视频或直播内容相关时，建议记录：

- 达人名称
- 内容链接
- 内容类型：短视频 / 直播 / 图文 / 联盟素材
- 是否使用了错误卖点
- 是否展示了非标配件
- 是否承诺了不确定效果
- 是否需要要求达人修改或下架内容
- 是否需要调整达人 brief

## 风险等级与影响范围

建议按风险等级判断处理优先级。

| 风险等级 | 判断标准 | 处理要求 |
| --- | --- | --- |
| 低 | 单个客户问题，不影响商品评价和平台绩效 | 正常处理并记录 |
| 中 | 多个客户出现相同反馈，可能影响评分或退货率 | 进入周报复盘，检查页面和供应链 |
| 高 | 涉及批量质量、合规、平台投诉或绩效风险 | 立即升级处理，必要时暂停推广或下架检查 |

## 处理结果与复盘结论

问题关闭后，需要记录最终结论，避免只记录处理动作、不记录复盘价值。

建议包含：

- 最终处理方式
- 客户是否接受
- 是否产生退款、退货、补发或赔付
- 是否影响店铺评分或绩效
- 根因判断
- 后续改进动作
- 是否需要同步到周报或商品测试复盘

## 高频问题统计

每周或每个测品周期建议统计：

- 售后问题总数
- 退款订单数
- 退货订单数
- 补发订单数
- 售后成本总额
- 高频问题类型 Top 3
- 高频 SKU
- 高频供应商
- 高频物流渠道
- 是否影响继续放量

## 下次预防动作

根据复盘结果，明确下一步预防动作。

可能动作包括：

- 修改商品标题或详情页
- 补充尺寸图、材质图、包装图或 FAQ
- 调整短视频脚本和字幕表达
- 更新达人 brief
- 修改质检标准
- 优化包装方案
- 更换物流渠道
- 调整供应商或压价
- 暂停上架、暂停推广或停止测试
