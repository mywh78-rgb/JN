# 商品图片拍摄 Brief

本文档用于规范跨境电商商品图片的拍摄、设计和交付要求。当前版本为基础版，适合用于商品上架、详情页制作、短视频素材准备和达人合作素材整理。

## 1. 文档用途

本 brief 用于帮助运营、摄影、设计和素材整理人员统一图片需求，减少返工，确保商品图片真实、清晰、合规，并能支持商品转化。

适用场景：

- 新商品上架图片准备
- 商品详情页素材补充
- 活动主图或广告图准备
- 达人寄样前素材说明
- 老商品图片优化

## 2. 商品图片需求概览

每个商品建议至少准备以下图片类型：

- 主图：用于商品列表和首屏展示。
- 细节图：展示材质、功能、接口、纹理和做工。
- 场景图：展示实际使用环境和目标人群。
- 尺寸图：说明商品尺寸、重量或容量。
- 包装图：展示包装、配件和到手内容。
- 对比图：展示使用前后或功能差异，需真实合规。

基础要求：

- 图片清晰、不模糊、不变形。
- 商品主体突出，信息表达明确。
- 图片内容必须与实际商品一致。
- 不使用未授权品牌、人物、图片或素材。

## 3. 主图拍摄要求

主图应快速传达商品是什么，并让买家清楚识别商品主体。

拍摄要求：

- 商品主体完整展示，不被裁切。
- 背景简洁，避免杂乱干扰。
- 光线充足，颜色接近实物。
- 重点 SKU 与实际售卖规格一致。
- 不添加误导性价格、夸张标签或虚假承诺。

建议输出：

- 白底主图
- 场景主图
- 商品组合主图
- 核心卖点主图

## 4. 细节图拍摄要求

细节图用于降低买家疑问，提升商品可信度。

建议展示：

- 材质纹理
- 功能按键或接口
- 开合、安装或使用细节
- 商品边角、缝线、做工
- 配件和组合内容
- 与手部或常见物品的比例参考

注意事项：

- 不过度修图导致与实物不符。
- 对可能影响购买决策的细节要清楚展示。
- 对容易误解的结构或配件要单独说明。

## 5. 场景图拍摄要求

场景图用于帮助买家理解商品适合在什么环境中使用。

建议场景：

- 家庭使用场景
- 办公或户外场景
- 节日、礼品或活动场景
- 目标人群使用场景
- 商品解决痛点的场景

拍摄原则：

- 场景应真实自然，不喧宾夺主。
- 商品应始终是画面重点。
- 场景和目标市场文化应匹配。
- 避免出现敏感、违规或侵权元素。

## 6. 尺寸图与包装图要求

尺寸图和包装图用于减少买家误解和售后争议。

尺寸图应包含：

- 长、宽、高或容量等关键参数。
- 必要时展示重量或可承载范围。
- 标注单位，避免单位混淆。
- 与详情页文字参数保持一致。

包装图应包含：

- 外包装展示。
- 包装内所有配件。
- 到手清单。
- 如有多个套装，应分别展示差异。

## 7. 禁止出现的内容

图片中不得出现可能导致违规、误导或侵权的内容。

禁止内容：

- 未授权品牌 Logo、商标或 IP 形象。
- 虚假认证、虚假奖项或无法证明的资质。
- 夸大功效、绝对化承诺或误导性对比。
- 与实际商品不一致的颜色、规格或配件。
- 平台禁止展示的敏感内容。
- 未获得授权的人像、音乐截图或第三方素材。

## 8. 图片交付检查清单

图片交付前建议逐项检查，确保可以直接用于上架或设计。

检查项：

- 是否包含主图、细节图、场景图、尺寸图和包装图。
- 图片是否清晰、完整、无明显色差。
- 商品规格是否与售卖 SKU 一致。
- 图片是否存在侵权或平台违规风险。
- 文案和参数是否与详情页一致。
- 文件命名是否清晰，方便后续管理。
- 是否已记录需要补拍或优化的素材。

建议命名方式：

```text
商品名_SKU_图片类型_序号
```

示例：

```text
portable-lamp_black_main_01.jpg
portable-lamp_black_detail_02.jpg
portable-lamp_black_size_03.jpg
```
