# 跨境电商供应链与核价执行 SOP

本文档用于 Temu、TikTok Shop、Amazon、Shopify 等平台的供货价、利润率、库存、补货、质检、物流、履约和风险判断。它承接选品 SOP，并在商品进入上架、履约和运营前确认供应链是否可执行。

## 一、文档用途

本 SOP 用于帮助运营人员判断一个商品是否具备稳定供货、合理利润、可控物流、可控售后和平台定价承接能力。

适用场景：

- 选品通过后的供应链核价
- 供应商报价对比
- 商品进入上架前的成本确认
- Temu 低价竞争力判断
- TikTok Shop 测品定价判断
- Amazon / Shopify 长期款定价承接
- 库存、补货、质检和履约风险判断
- 是否进入测试、压价或暂缓决策

使用建议：

- 先确认供应商基础资料和报价来源。
- 再核验供货价、物流成本、包装成本和售后风险。
- 根据平台目标价测算毛利率和毛利额。
- 对库存、质检、资质和履约风险做最终判断。
- 输出供应链结论，决定进入测试、继续压价或暂缓。

## 四、供货价核验

供货价不能只看供应商口头报价，应核验价格包含什么、不包含什么。

### 1. 核验内容

- 商品单价是否含包装。
- 是否包含配件、说明书、标签或贴标。
- 是否含国内运费。
- 是否含质检、换包装或定制费用。
- 是否有阶梯价和 MOQ。
- 是否有颜色、尺码、套装差异价。
- 是否有临时涨价或缺货风险。

## 五、核价流程

核价应从商品总成本出发，而不是只看供货价。

### 1. 基础公式

```text
总成本 = 供货价 + 包装成本 + 国内运费 + 质检成本 + 国际物流成本 + 平台费用 + 预估售后成本
```

```text
毛利额 = 售价 - 总成本
```

```text
毛利率 = 毛利额 / 售价
```

### 2. 核价步骤

1. 确认供货价和阶梯价。
2. 确认包装、配件、标签和说明书成本。
3. 估算国内运费和国际物流成本。
4. 估算平台费用、佣金、广告或达人佣金。
5. 估算退货、破损、售后和补发成本。
6. 按目标平台售价测算毛利率和毛利额。
7. 判断是否进入测试、压价或暂缓。

## 六、毛利率与毛利额测算

毛利率代表利润健康度，毛利额代表每单是否值得做。

### 1. 毛利率判断

| TikTok Shop | 待填 | 待填 | 待填 | 待填 | 待定 |
| Temu | 待填 | 待填 | 待填 | 待填 | 待定 |
| Amazon | 待填 | 待填 | 待填 | 待填 | 待定 |
| Shopify | 待填 | 待填 | 待填 | 待填 | 待定 |

### 2. 判断原则

- 毛利率过低：难以承受佣金、广告、退货和折扣。
- 毛利额过低：即使毛利率看起来不错，每单实际利润也可能不值得做。
- 高售价商品：需要考虑转化难度、信任成本和售后成本。
- 低价商品：需要重点检查物流和包装是否吃掉利润。

## 七、物流成本与体积重量判断

物流成本是跨境商品能否成立的重要因素。

### 1. 必须确认的信息

- 商品净重
- 包装后重量
- 商品尺寸
- 包装后尺寸
- 是否抛货
- 是否易碎
- 是否带电、带磁、液体、粉末或特殊属性
- 是否需要加固包装
- 是否适合海外仓或平台仓

### 2. 体积重量判断

```text
体积重量 = 长 x 宽 x 高 / 计泡系数
```

不同物流渠道计泡系数可能不同，应以实际渠道为准。

### 3. 高风险物流特征

- 体积大、售价低。
- 易碎、易漏、易变形。
- 包装成本高。
- 带电、带磁、液体、粉末等特殊属性。
- 清关或平台入仓限制较多。

## 八、库存与补货判断

库存和补货能力决定商品能否承接内容、达人和平台流量。

### 1. 库存检查

- 当前现货数量。
- 可锁库存数量。
- 补货周期。
- 生产周期。
- 供应商断货风险。
- 多 SKU 是否库存均衡。
- 热销 SKU 是否容易缺货。

### 2. 补货判断

建议关注：

- 首批测试量是否合理。
- 是否可以小批量多次补货。
- 爆单后是否能快速补货。
- 是否有替代供应商。
- 是否会因为缺货影响店铺评分或达人合作。

## 九、质检与出货检查

质检是减少退货、差评和售后风险的关键动作。

### 1. 质检内容

- 商品外观是否有破损、污渍、色差。
- 颜色、尺码、规格是否与订单一致。
- 配件是否齐全。
- 包装是否完整。
- 标签、说明书、条码是否正确。
- 功能是否正常。
- 是否存在异味、松动、划痕或质量瑕疵。

### 2. 出货检查

- SKU 是否匹配。
- 数量是否正确。
- 包装是否适合跨境运输。
- 是否需要拍照留档。
- 是否需要加固包装。
- 是否需要贴标或换包装。

## 十、Temu 低价竞争力判断

Temu 更强调价格竞争力和履约成本控制。

判断要点：

- 供货价是否足够低。
- 是否容易被同类商品压价。
- 包装和物流成本是否可控。
- 是否适合低价走量。
- 是否有规格、组合或包装差异化空间。
- 退货和售后成本是否会吞掉利润。

建议结论：

- 具备低价优势：可进入 Temu 测试。
- 价格接近竞品：需要继续压价或做差异化。
- 成本明显偏高：暂缓 Temu。

## 十一、TikTok Shop / Amazon / Shopify 定价承接

不同平台的定价承接逻辑不同。

### 1. TikTok Shop

重点看：

- 售价是否适合短视频冲动购买。
- 是否能承受达人佣金和优惠券。
- 是否能承接直播专属价。
- 是否有足够库存应对内容爆发。

### 2. Amazon

重点看：

- 是否能承受广告成本和平台费用。
- 是否有足够毛利用于长期评价沉淀。
- 是否能用 A+图和详情页解释价格。
- 是否能控制退货和差评。

### 3. Shopify

重点看：

- 是否有品牌溢价空间。
- 是否能承受广告获客成本。
- 是否适合组合销售、加购或复购。
- 是否能通过内容和详情页建立信任。

## 十二、退货、破损、售后风险判断

供应链核价必须预估售后成本。

高风险来源：

- 尺码复杂。
- 颜色和图片容易有差异。
- 易碎或易损。
- 使用方法复杂。
- 材质容易不符合预期。
- 商品效果难验证。
- 包装保护不足。
- 买家容易误解尺寸、功能或配件。

判断动作：

- 高退货风险商品要提高预估售后成本。
- 易碎商品要增加包装成本。
- 尺码商品要补充尺码图和说明。
- 功能复杂商品要补充说明书和短视频演示。

## 十三、合规与资质风险检查

商品进入供应链核价时，也要确认资质和合规风险。

检查内容：

- 是否涉及品牌授权。
- 是否涉及专利、外观或版权风险。
- 是否需要认证、检测报告或安全标签。
- 是否属于平台禁售或限售类目。
- 是否涉及医疗、功效、安全、儿童、母婴、食品接触、宠物等敏感领域。
- 供应商是否能提供资质文件。
- 商品包装和说明是否符合目标市场要求。

高风险商品应暂缓，直到资质和规则确认清楚。

## 十四、是否进入测试 / 压价 / 暂缓的判断标准

### 1. 进入测试

满足以下条件可进入测试：

- 供货价清楚。
- 毛利率和毛利额可接受。
- 物流成本可控。
- 库存和补货稳定。
- 质检和包装可执行。
- 合规风险低或可控。
- 能承接目标平台定价。

### 2. 继续压价

出现以下情况建议压价：

- 商品有潜力但供货价偏高。
- Temu 或低价平台竞争压力大。
- 达人佣金、广告或优惠后利润不足。
- 供应商报价明显高于同类渠道。

### 3. 暂缓

出现以下情况建议暂缓：

- 总成本不清楚。
- 毛利率过低。
- 物流成本不可控。
- 库存不稳定。
- 质检不可控。
- 合规或资质风险不清楚。
- 退货和售后风险过高。
