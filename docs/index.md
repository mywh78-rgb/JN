# 文档导航

本文档是 `JN` 仓库中所有 SOP 和跨境电商业务文档的总导航，用于帮助快速找到对应流程、模板和测试记录。

## 1. 文档导航说明

`docs/` 目录用于存放跨境电商项目相关的操作文档、流程规范和业务 SOP。

当前文档覆盖：

- Codex 设置与工作流
- TikTok Shop 日常运营
- 商品上架与订单履约
- 内容计划与短视频脚本
- 达人联盟 BD
- 商品图片素材 brief

## 2. 基础设置文档

- `codex-setup.md`：记录 Codex 连接 GitHub、读取仓库、创建分支、提交文件和创建 Pull Request 的基础流程。

适合在开始维护仓库前阅读。

## 3. 店铺运营 SOP

- `tiktok-shop-sop.md`：TikTok Shop 日常运营 SOP，覆盖店铺检查、商品状态、订单履约、内容短视频、达人联盟和风险合规。

适合用作日常运营检查清单。

## 4. 商品与订单文档

- `product-listing-guide.md`：商品上架资料规范，覆盖标题、图片、卖点、价格、库存、规格和合规检查。
- `order-workflow.md`：订单处理流程，覆盖订单状态、新订单处理、发货物流、退款退货、异常订单和每日复盘。

适合用于商品发布、订单履约和售后处理。

## 5. 内容与营销文档

- `content-calendar.md`：内容计划日历，覆盖内容规划周期、每周安排、短视频选题、商品匹配、发布状态和数据复盘。
- `tiktok-video-script-sop.md`：TikTok 短视频脚本 SOP，覆盖脚本结构、Hook、卖点表达、场景设计、CTA、分镜和发布前检查。

适合用于短视频内容计划和脚本生产。

## 6. 达人与联盟文档

- `creator-affiliate-bd.md`：达人联盟 BD 流程，覆盖达人筛选、联系话术、样品寄送、合作跟进、联盟佣金和效果复盘。

适合用于达人开发、联盟推广和合作管理。

## 7. 素材与脚本文档

- `product-image-brief.md`：商品图片拍摄 Brief，覆盖主图、细节图、场景图、尺寸图、包装图、禁止内容和交付检查清单。
- `tiktok-video-script-sop.md`：TikTok 短视频脚本 SOP，可与图片 brief、内容日历和达人 brief 配合使用。

适合用于图片素材准备、拍摄沟通和短视频制作。

## 8. 推荐使用顺序

建议按以下顺序阅读和使用文档：

1. `codex-setup.md`：先了解仓库和 Codex 工作流。
2. `tiktok-shop-sop.md`：建立店铺日常运营主流程。
3. `product-listing-guide.md`：准备商品资料并完成上架检查。
4. `order-workflow.md`：处理订单履约和售后异常。
5. `content-calendar.md`：规划每周内容节奏。
6. `product-image-brief.md`：准备商品图片和素材需求。
7. `tiktok-video-script-sop.md`：产出短视频脚本和分镜。
8. `creator-affiliate-bd.md`：推进达人和联盟合作。
