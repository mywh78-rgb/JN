# 商品上架资料规范

本文档用于规范跨境电商商品上架前后的资料准备、信息填写和合规检查流程。当前版本为基础版，适合后续结合具体平台规则继续扩展。

## 1. 文档用途

本指南用于帮助运营人员在商品上架前统一准备资料，减少信息缺失、图片不合规、价格错误和审核失败等问题。

适用场景：

- 新商品首次上架
- 已上架商品信息优化
- 商品资料复查
- 多平台商品资料整理

## 2. 商品上架前准备

上架前应先确认商品基础信息完整、真实、可追溯。

- 确认商品名称、类目、品牌和型号。
- 准备商品主图、细节图、包装图和使用场景图。
- 整理商品规格、颜色、尺寸、材质、重量和包装信息。
- 确认库存数量、采购成本、建议售价和利润空间。
- 检查商品是否涉及平台禁售、限售或资质要求。
- 准备售后说明、保修信息和常见问题回答。

## 3. 商品标题规范

商品标题应清晰表达商品核心信息，避免夸大和堆砌关键词。

建议包含：

- 核心品类词
- 主要卖点
- 适用场景
- 关键规格
- 目标人群或用途

注意事项：

- 不使用虚假品牌词或侵权词。
- 不堆砌无关关键词。
- 不使用平台禁止的夸张表达。
- 保持标题可读，避免过长或重复。

## 4. 商品图片规范

商品图片应真实、清晰，并能准确展示商品状态。

建议准备：

- 主图：突出商品主体，背景简洁。
- 细节图：展示材质、功能、接口、尺寸等细节。
- 场景图：展示商品使用方式和适用场景。
- 包装图：展示包装内容和配件清单。
- 尺寸图：标注关键尺寸，减少买家误解。

注意事项：

- 不使用未经授权的图片。
- 不展示与实际商品不符的内容。
- 不加入误导性水印、价格或承诺。
- 图片应符合平台尺寸和清晰度要求。

## 5. 商品卖点与详情页规范

商品卖点应围绕买家决策重点展开，详情页要完整解释商品价值。

建议包含：

- 商品核心功能
- 主要使用场景
- 与同类商品的差异点
- 材质、尺寸、重量和包装信息
- 使用方法和注意事项
- 售后说明和常见问题

写作原则：

- 使用真实、可验证的描述。
- 避免夸大功效或承诺结果。
- 用清晰结构帮助买家快速理解。
- 对容易误解的规格或限制提前说明。

## 6. 价格、库存与规格设置

价格、库存和规格会直接影响转化、履约和售后风险。

检查事项：

- 售价是否覆盖成本、平台费用、物流费用和促销折扣。
- 库存数量是否真实可售。
- SKU 规格是否与图片和详情页一致。
- 颜色、尺寸、套装数量等选项是否清晰。
- 活动价格和日常价格是否设置正确。
- 缺货商品是否及时下架或调整库存。

## 7. 合规检查清单

上架前应完成基础合规检查，避免商品审核失败或产生处罚。

- 是否涉及禁售或限售类目。
- 是否需要品牌授权、认证或资质文件。
- 商品标题、图片和详情页是否存在侵权内容。
- 是否使用未经授权的商标、人物、音乐、图案或素材。
- 是否存在虚假宣传、功效夸大或误导性承诺。
- 是否符合目标市场的安全、标签和包装要求。

## 8. 上架后复查事项

商品发布后，应及时检查前台展示和运营数据。

- 确认商品页面可以正常打开。
- 检查标题、图片、价格、库存和规格展示是否正确。
- 查看商品是否通过审核或存在平台提示。
- 检查折扣、优惠券和活动展示是否正常。
- 观察曝光、点击、加购和转化数据。
- 记录需要优化的问题，并安排后续更新。
