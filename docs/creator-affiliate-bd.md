# 达人联盟 BD 流程

本文档用于记录 TikTok Shop 及跨境电商场景中的达人筛选、联系、样品寄送、合作跟进和效果复盘流程。当前版本为基础版，适合后续结合真实达人数据和平台规则继续扩展。

## 1. 文档用途

本流程用于帮助运营人员系统管理达人和联盟合作，提升沟通效率，减少样品浪费，并沉淀可复用的合作判断标准。

适用场景：

- 达人名单筛选
- 达人私信或邮件开发
- 样品寄送和内容跟进
- 联盟商品推广管理
- 达人合作效果复盘

## 2. 达人筛选标准

筛选达人时，应优先关注内容匹配度和转化潜力，而不只看粉丝数量。

建议评估维度：

- 内容类目是否与商品匹配。
- 受众人群是否符合目标买家画像。
- 近期内容更新是否稳定。
- 视频播放、互动和评论质量是否真实。
- 是否有带货经验或商品展示能力。
- 过往合作内容是否自然、可信。
- 是否存在违规、争议或低质量内容风险。

基础分层：

- 优先合作：内容匹配、互动真实、转化潜力高。
- 可测试：数据一般但内容风格适合。
- 暂不合作：类目不匹配、更新不稳定或风险较高。

## 3. 达人联系话术

联系达人时，应简洁说明品牌、商品、合作形式和下一步动作。

基础私信模板：

```text
Hi [Name], we like your content about [category/topic].
We are looking for creators to try and share our [product name].
Would you be interested in a sample collaboration or affiliate promotion?
If yes, I can send more product details and commission information.
```

中文记录模板：

- 达人名称：
- 平台主页：
- 联系方式：
- 联系时间：
- 商品方向：
- 合作形式：样品 / 佣金 / 付费 / 混合
- 当前状态：待回复 / 已回复 / 已寄样 / 已发布 / 已结束

注意事项：

- 不夸大收益，不承诺无法保证的结果。
- 不频繁骚扰同一达人。
- 保留沟通记录，方便后续跟进。

## 4. 样品寄送流程

样品寄送前应确认合作意向、收件信息和内容要求。

流程步骤：

1. 确认达人愿意合作。
2. 发送商品信息、卖点和合作要求。
3. 收集收件人、电话、地址和邮编。
4. 确认样品 SKU、数量和库存。
5. 安排发货并记录物流单号。
6. 跟进签收状态。
7. 达人签收后提醒内容发布时间。

记录字段：

- 达人名称
- 寄送商品
- 寄送日期
- 物流单号
- 签收状态
- 预计发布时间
- 跟进备注

## 5. 合作内容跟进

样品寄出后，应持续跟进内容产出，避免样品寄出后无反馈。

跟进节点：

- 样品发出后通知达人。
- 物流签收后确认是否收到。
- 签收 3 到 5 天后跟进试用反馈。
- 内容发布前确认商品链接或联盟入口。
- 内容发布后记录链接和数据。
- 对表现好的内容安排二次合作。

内容检查：

- 商品展示是否清楚。
- 卖点表达是否准确。
- 是否包含正确商品链接或购物入口。
- 是否存在违规、侵权或夸大宣传。
- 是否适合复用为广告素材或店铺内容。

## 6. 联盟佣金设置

联盟佣金应兼顾达人积极性、商品利润和活动目标。

设置建议：

- 根据商品毛利设置佣金上限。
- 对重点推广商品设置更有吸引力的佣金。
- 对新品测试期可以设置短期激励。
- 定期检查佣金是否影响利润。
- 避免佣金设置错误导致亏损。

记录内容：

- 商品名称
- SKU
- 日常佣金比例
- 活动佣金比例
- 生效时间
- 结束时间
- 适用达人或联盟计划

## 7. 达人效果复盘

达人合作后应复盘内容表现和转化效果，决定是否继续合作。

复盘指标：

- 内容发布时间
- 播放量
- 互动量
- 商品点击量
- 加购量
- 订单量
- 成交金额
- 退款或售后情况
- 合作成本

复盘结论：

- 是否适合长期合作。
- 是否适合寄送更多样品。
- 是否适合提高佣金或付费合作。
- 哪些内容角度值得复用。
- 哪些问题需要下次避免。

## 8. 风险与注意事项

达人合作涉及品牌、合规和成本风险，应提前控制。

注意事项：

- 不与明显违规、刷量或争议达人合作。
- 不使用未经授权的达人内容做广告投放。
- 样品寄送前确认商品和库存真实可用。
- 对合作要求、发布时间和内容使用权提前沟通。
- 不承诺虚假收益或平台不允许的推广方式。
- 对高成本合作保留确认记录和复盘结果。
