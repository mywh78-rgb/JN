# 跨境电商选品运营执行 SOP

本文档用于 TikTok Shop、Temu、Amazon、Shopify 等平台的商品筛选、利润测算、风险判断和上架优先级决策。它是商品进入上架、图片、短视频、达人和运营流程之前的选品决策入口。

## 一、文档用途

本 SOP 用于帮助运营人员在选品阶段判断一个商品是否值得测试、适合哪个平台、风险是否可控，以及是否适合新手或小团队执行。

适用场景：

- 新品筛选
- 平台选品池整理
- TikTok Shop 测品决策
- Temu 低价竞争力判断
- Amazon / Shopify 长期款判断
- 商品利润和物流成本测算
- 退货风险、合规风险和履约风险判断
- 上架优先级排序

使用建议：

- 先收集选品基础资料。
- 再使用 100 分评分模型打分。
- 根据分数进行 S/A/B/C/D 商品分级。
- 针对不同平台补充专项判断。
- 最后输出选品结论和下一步动作。

## 二、适用平台与选品目标

不同平台的选品目标不同，应避免用同一套标准判断所有商品。

| 平台 | 选品目标 | 重点判断 |
| --- | --- | --- |
| TikTok Shop | 找到适合短视频、达人和自然流测试的商品 | 视觉卖点、3 秒 Hook、内容传播、达人适配 |
| Temu | 找到低价竞争力强、履约成本可控的商品 | 成本、价格带、物流体积、供应链稳定 |
| Amazon | 找到可长期沉淀搜索、评价和利润的商品 | 搜索需求、评价空间、差异化、广告承接 |
| Shopify | 找到适合品牌化、独立站承接和内容转化的商品 | 品牌故事、复购、素材表现、转化路径 |

选品目标应明确：

- 是为了快速测品，还是长期经营。
- 是为了自然流内容，还是搜索和广告承接。
- 是为了低价冲量，还是高毛利品牌化。
- 是为了新手低风险起步，还是成熟团队放大测试。

## 三、选品基础资料清单

选品前应先收集基础资料，避免只凭感觉判断。

| 项目 | 检查内容 | 状态 |
| --- | --- | --- |
| 商品名称 | 商品是什么，是否容易一句话讲清楚 | 待确认 |
| 类目 | 属于哪个平台类目，是否有禁售或限售风险 | 待确认 |
| 目标平台 | TikTok Shop / Temu / Amazon / Shopify | 待确认 |
| 目标人群 | 谁会买，使用场景是什么 | 待确认 |
| 核心卖点 | 商品解决什么问题，有什么第一眼亮点 | 待确认 |
| 成本价 | 采购成本、包装成本、质检成本 | 待确认 |
| 预估售价 | 不同平台的目标售价和价格带 | 待确认 |
| 毛利额 | 单件商品可获得的毛利金额 | 待确认 |
| 毛利率 | 毛利额占售价比例 | 待确认 |
| 物流成本 | 头程、尾程、仓储、体积重量成本 | 待确认 |
| 退货风险 | 尺码、易损、质量、预期不符等风险 | 待确认 |
| 合规风险 | 资质、认证、侵权、禁售、敏感功效 | 待确认 |
| 内容素材 | 是否容易拍短视频、图片、A+图和达人内容 | 待确认 |
| 供应链 | 是否稳定供货，是否容易补货 | 待确认 |
| 竞品情况 | 是否已有同类产品，竞争强度如何 | 待确认 |

## 四、100 分选品评分模型

建议使用 100 分模型对商品进行初筛。评分不是绝对结论，但可以帮助统一判断标准。

| 评分维度 | 分值 | 判断重点 |
| --- | --- | --- |
| 市场需求 | 15 分 | 是否有明确人群、场景和购买需求 |
| 内容传播潜力 | 15 分 | 是否适合短视频、3 秒 Hook、达人展示 |
| 利润空间 | 15 分 | 毛利率、毛利额、价格带是否健康 |
| 履约难度 | 10 分 | 是否轻小、易包装、物流成本可控 |
| 退货风险 | 10 分 | 是否容易尺码不符、破损、误解或客诉 |
| 合规风险 | 10 分 | 是否涉及资质、侵权、禁售、功效风险 |
| 平台适配度 | 10 分 | 是否适合 TikTok Shop、Temu、Amazon 或 Shopify |
| 供应链稳定性 | 8 分 | 是否稳定供货、可补货、质量可控 |
| 素材制作难度 | 7 分 | 是否容易制作图片、视频和达人内容 |

### 评分说明

- 80 分以上：具备较强测试或推进价值。
- 65 到 79 分：可以小规模测试，但需要关注风险点。
- 50 到 64 分：谨慎测试，优先补充信息或优化供应链。
- 50 分以下：不建议当前阶段推进。

## 五、S/A/B/C/D 商品分级规则

根据评分、风险和平台适配度进行分级。

### S 级：优先测试 / 重点推进

特征：

- 评分 85 分以上。
- 需求明确，内容传播强。
- 利润空间健康。
- 履约和合规风险低。
- 有清晰平台打法和素材方向。

建议动作：

- 优先进入上架准备。
- 准备图片、视频、达人 brief。
- 安排小批量测品或重点推广。

### A 级：可以测试

特征：

- 评分 75 到 84 分。
- 有明确卖点和目标人群。
- 风险可控，但需要验证内容或价格。

建议动作：

- 小规模测试。
- 先做主图、短视频和基础上架。
- 观察点击、加购、转化和退货反馈。

### B 级：谨慎测试

特征：

- 评分 65 到 74 分。
- 存在利润、履约、素材或合规方面的不确定点。

建议动作：

- 暂缓大规模投入。
- 先补充竞品调研和成本测算。
- 仅在资源允许时小样本测试。

### C 级：暂缓

特征：

- 评分 50 到 64 分。
- 卖点不清、利润一般、风险较多。

建议动作：

- 暂不进入上架流程。
- 等供应链、素材、价格或合规问题明确后再评估。

### D 级：不建议做

特征：

- 评分低于 50 分。
- 合规风险高、利润低、履约困难或退货风险高。

建议动作：

- 不进入测试。
- 不投入图片、视频和达人资源。

## 六、TikTok Shop 自然流潜力判断

TikTok Shop 选品应重点判断内容传播和自然流转化潜力。

### 1. 适合 TikTok Shop 的商品特征

- 3 秒内能看懂商品用途。
- 有明显视觉卖点或使用反差。
- 能通过短视频展示使用过程。
- 适合达人自然分享或 UGC 展示。
- 容易引发评论、收藏、点击或转发。
- 价格不需要太长决策周期。
- 商品链接与视频场景容易匹配。

### 2. 需要谨慎的商品

- 需要长篇解释才能理解。
- 画面表现力弱，短视频难拍。
- 涉及强资质、医疗、功效或安全承诺。
- 客单高但信任基础不足。
- 容易因尺码、材质或效果预期产生退货。

### 3. 判断问题

- 这个商品能不能在前 3 秒吸引停留？
- 有没有真实痛点或强反差？
- 达人是否愿意自然展示？
- 商品是否适合挂车转化？
- 是否能拆出 3 到 5 个短视频选题？

## 七、Temu 低价竞争力与履约成本判断

Temu 场景下，低价竞争力和履约成本是核心判断点。

### 1. 价格竞争力

- 是否有足够低的供应链成本。
- 是否能承受平台低价竞争。
- 是否容易被同质化竞品压价。
- 是否有包装、组合、规格或功能差异化空间。

### 2. 履约成本

- 商品是否轻小。
- 体积重量是否可控。
- 是否易碎、易漏、易变形或需要特殊包装。
- 包装成本是否会吃掉利润。
- 退货和售后成本是否可控。

### 3. 谨慎商品

- 体积大但客单低。
- 易碎、易坏、易漏液。
- 尺码复杂、颜色误差大。
- 质量稳定性差。
- 极易被低价竞品替代。

## 八、Amazon / Shopify 长期款判断

Amazon 和 Shopify 更适合判断商品是否具备长期经营价值。

### 1. Amazon 长期款判断

重点看：

- 是否有稳定搜索需求。
- 是否适合沉淀评价。
- 是否能通过详情页和 A+图解释价值。
- 是否具备差异化卖点。
- 是否有广告承接能力。
- 是否能控制退货和差评。

### 2. Shopify 长期款判断

重点看：

- 是否适合品牌故事。
- 是否能通过内容、广告或达人引流。
- 是否有复购、组合销售或加购空间。
- 是否能做独立站详情页和邮件营销。
- 是否有清晰目标人群和使用场景。

### 3. 长期款谨慎信号

- 只能靠低价成交。
- 没有差异化和品牌化空间。
- 售后解释成本高。
- 容易被平台规则或资质限制。
- 图片和视频很难证明价值。

## 九、利润率、毛利额、物流成本、退货风险、合规风险判断

选品必须同时看利润和风险，不能只看售价或热度。

### 1. 利润率判断

建议计算：

```text
毛利率 = (售价 - 商品成本 - 平台费用 - 物流成本 - 包装成本 - 预估售后成本) / 售价
```

判断要点：

- 毛利率太低时，无法承受佣金、广告和退货。
- 高毛利商品也要检查是否有合规或售后风险。
- 不同平台对毛利率要求不同，应按平台单独测算。

### 2. 毛利额判断

毛利额决定每单是否值得做。

检查问题：

- 单件毛利是否足够覆盖运营和推广成本？
- 是否能承受达人佣金或直播专属价？
- 是否能承受优惠券、折扣和退货损失？

### 3. 物流成本判断

重点看：

- 重量
- 体积
- 包装方式
- 是否易碎
- 是否需要特殊仓储或运输
- 是否适合跨境运输

### 4. 退货风险判断

高退货风险通常来自：

- 尺码复杂
- 颜色误差
- 材质预期不符
- 功能效果难验证
- 易损坏
- 安装或使用复杂
- 买家容易误解商品大小或用途

### 5. 合规风险判断

高合规风险通常来自：

- 侵权品牌或外观
- 未授权图片或 IP
- 医疗、功效、安全类敏感表达
- 儿童、食品接触、宠物、母婴、健康相关资质
- 禁售或限售类目
- 虚假认证或无法证明的资质

## 十、新手/小白友好商品判断标准

适合新手或小团队的商品，应优先低风险、易解释、易履约、易出内容。

### 1. 新手友好特征

- 轻小件，物流成本可控。
- 不易碎，不易漏，不易变形。
- 不涉及复杂尺码。
- 不涉及高风险资质。
- 使用方法简单，容易拍短视频。
- 图片和视频能清楚展示卖点。
- 客诉点少，退货风险低。
- 供应链稳定，补货容易。
- 价格不太高，买家决策门槛低。

### 2. 新手谨慎商品

- 泳装、服装、鞋类等尺码风险较高的商品。
- 医疗、健康、母婴、安全承诺强的商品。
- 易碎、液体、带电、带磁或大件商品。
- 需要复杂安装或专业知识的商品。
- 侵权风险高、外观相似度高的商品。

### 3. 新手判断问题

- 我能不能用一句话说清楚它解决什么问题？
- 我能不能拍出 3 条短视频？
- 它的退货原因是否容易预防？
- 它的物流成本是否可控？
- 它是否需要资质或复杂售后？
