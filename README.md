# JN

JN 是一个跨境电商 Codex 测试仓库，用于学习和验证 Codex 如何读取、整理、维护 GitHub 项目文件。

当前仓库重点不是实现业务系统，而是沉淀一套完整的跨境电商 SOP 文档体系，覆盖 SKU 项目总控、竞品调研、选品、供应链核价、商品上架、视觉素材、短视频、达人合作、内容日历、订单履约、售后问题、客服回复、促销复盘、风险合规、店铺运营、商品测试复盘、店铺周报、月度经营复盘和 Codex 测试过程。

## 项目简介

这个仓库用于模拟一个跨境电商项目的文档维护流程。通过多轮 Codex 测试，仓库逐步建立了 `docs/` 业务 SOP 和 `tasks/` 测试记录两类内容。

## 仓库用途

- 学习 Codex 读取 GitHub 仓库内容
- 练习通过分支和 Pull Request 维护文件
- 整理跨境电商运营 SOP、模板和工作流
- 记录每一轮 Codex 测试任务和结果
- 为后续扩展前端、后端或自动化工具预留文档基础

## 当前文档体系用途说明

当前文档体系可以用于一个跨境电商 SKU 从立项到复盘的完整流程：先通过 SKU 总控入口明确项目状态，再完成市场调研、选品判断、供应链核价、商品上架、素材制作、短视频脚本、达人合作、内容排期、日常运营、订单履约、售后记录、促销复盘、风险合规和经营复盘。

适用平台包括 TikTok Shop、Amazon、Shopify、Temu，也可作为其他跨境电商平台的运营文档基础。

## docs 文档说明

- `docs/index.md`：所有 SOP 和业务文档的总导航
- `docs/codex-setup.md`：Codex 设置和使用流程记录
- `docs/sku-launch-master-sop.md`：SKU 单品项目总控交付 SOP，用于串联单个 SKU 从选品到测试复盘和周报沉淀的完整流程
- `docs/market-competitor-research-sop.md`：竞品与市场调研 SOP，用于选品、核价、上架、图片、视频、达人和供应链判断前的调研入口
- `docs/product-selection-sop.md`：跨境电商选品运营执行 SOP
- `docs/supply-chain-pricing-sop.md`：供应链与核价执行 SOP
- `docs/product-listing-guide.md`：美国 TikTok Shop 商品上架执行 SOP
- `docs/product-image-brief.md`：商品图片视觉执行 SOP
- `docs/tiktok-video-script-sop.md`：TikTok 带货短视频脚本执行 SOP
- `docs/creator-affiliate-bd.md`：美国 TikTok Shop 达人联盟 BD 执行 SOP
- `docs/content-calendar.md`：内容计划日历
- `docs/tiktok-shop-sop.md`：TikTok Shop 日常运营执行 SOP
- `docs/order-workflow.md`：订单处理流程
- `docs/after-sales-issue-log-template.md`：售后问题记录规范模板，用于记录、分类、跟进和复盘退款、退货、补发、赔付、质量、物流和内容误导等售后问题
- `docs/customer-service-reply-template.md`：客服回复模板，用于售前咨询、物流问题、退货退款、产品问题、差评安抚、售后跟进和合规话术
- `docs/product-test-review-template.md`：商品测试复盘模板，用于沉淀测品结果和决策闭环
- `docs/store-weekly-report-template.md`：店铺周报模板，用于每周复盘多平台店铺运营表现
- `docs/promotion-campaign-review-template.md`：促销活动复盘模板，用于复盘优惠券、限时折扣、达人专属价、直播专属价、平台活动和广告活动
- `docs/risk-compliance-log-template.md`：风险合规记录模板，用于记录商品、素材、内容、达人、广告、平台处罚、敏感词、资质、侵权和虚假宣传等风险
- `docs/monthly-business-review-template.md`：月度经营复盘模板，用于汇总选品、供应链、上架、素材、内容、达人、订单、售后、利润、风险和下月动作

## 推荐阅读顺序

1. 先阅读 `README.md`，了解仓库用途和整体结构。
2. 阅读 `docs/index.md`，查看完整业务流程导航。
3. 阅读 `docs/codex-setup.md`，了解 Codex 工作流。
4. 阅读 `docs/sku-launch-master-sop.md`，作为单个 SKU 项目从启动到复盘的总控入口。
5. 阅读 `docs/market-competitor-research-sop.md`，完成竞品、市场、趋势、评论、视觉、视频、价格和合规调研。
6. 阅读 `docs/product-selection-sop.md`，判断商品是否值得做。
7. 阅读 `docs/supply-chain-pricing-sop.md`，确认供货价、利润、物流、库存、质检和履约风险。
8. 阅读 `docs/product-listing-guide.md`，准备商品上架资料和审核前检查。
9. 阅读 `docs/product-image-brief.md`，规划商品主图、A+图、场景图和视觉素材。
10. 阅读 `docs/tiktok-video-script-sop.md`，规划短视频脚本、英文字幕、Hook 和 AI 视频提示词。
11. 阅读 `docs/creator-affiliate-bd.md`，推进达人建联、联盟营销、样品和直播合作。
12. 阅读 `docs/content-calendar.md`，安排内容排期和复盘。
13. 阅读 `docs/tiktok-shop-sop.md`，执行 TikTok Shop 日常运营检查和每周复盘。
14. 阅读 `docs/order-workflow.md`，处理订单履约和售后异常。
15. 阅读 `docs/after-sales-issue-log-template.md`，记录售后问题、处理动作、责任归因和复盘结论。
16. 阅读 `docs/customer-service-reply-template.md`，统一售前、售中、售后和差评安抚回复口径。
17. 阅读 `docs/product-test-review-template.md`，完成商品测试复盘并判断继续放量、优化再测、暂缓或停止测试。
18. 阅读 `docs/store-weekly-report-template.md`，汇总店铺周度运营表现和下周重点动作。
19. 阅读 `docs/promotion-campaign-review-template.md`，复盘优惠券、折扣、达人专属价、直播专属价、平台活动和广告活动。
20. 阅读 `docs/risk-compliance-log-template.md`，记录和追踪商品、内容、广告、达人和平台处罚相关风险。
21. 阅读 `docs/monthly-business-review-template.md`，完成月度经营复盘和下月动作规划。
22. 阅读 `tasks/` 下的测试记录，了解每一轮 Codex 操作历史。
