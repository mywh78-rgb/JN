# JN

JN 是一个跨境电商 Codex 测试仓库。

这个仓库用于学习和验证 Codex 如何读取、整理和维护项目文件。当前阶段重点不是实现业务系统，而是建立清晰、可维护的项目文档结构，方便后续记录操作流程、测试结果和任务变更。

## 仓库用途

- 学习 Codex 读取 GitHub 仓库内容
- 练习让 Codex 整理项目目录和文档
- 记录跨境电商项目相关的测试任务
- 为后续添加前端、后端或业务文档预留基础结构

## 当前目录结构

```text
JN/
├── README.md
├── .gitignore
├── docs/
│   └── codex-setup.md
└── tasks/
    └── first-test.md
```

## 目录说明

- `docs/`：存放操作文档、设置说明和流程记录
- `tasks/`：存放任务记录、测试结果和阶段总结
- `.gitignore`：记录基础忽略规则，避免提交本地临时文件
